Grid 布局和 flex 布局

区别:

  1. 一维布局: flex;二维布局: grid;
  2. 从内容出发用 flex,布局出发用 grid.

内容出发:先有一组数据,数量不固定,希望均匀分布在容器中,由内容自己的大小占据空间.
布局出发: 先规划网格,再把元素往里面填充.
